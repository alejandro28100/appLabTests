## About
Admin Dashboard of courses website.
## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies Used

 - React with Next js
 - Typescript
 - Tailwind CSS
 - Headless UI
 - Vercel (deployment)
